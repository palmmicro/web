EP668 IP Phone
  
KITTY GW-EP668/A IP Phone manufactured by IPGW uses its own software slightly different from our PA168S reference design. The operation of the phone is almost the same with the standard design.
Standard reference design IP phone web manual: SIP IAX2 H323 MGCP. Menu manual: SIP IAX2 H323 MGCP. Telnet manual: SIP IAX2 H323 MGCP.
EP668 uses its own software with name like ep668_sip_us_169003.bin. Although it looks like YWH10, Primeworx P100 and IPH_E00_2G, the only common thing about those phones is that they all use perhaps the cheapest PSTN phone case available in the market.
You can restore password if you do not know the password of the phone. You may find in the debug window of Palmtool that the IP address in safe mode is 192.168.0.100 instead of our standard 192.168.1.100.
