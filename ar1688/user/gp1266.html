GP1266 and GP1260 IP Phones
  
Digitmat Technology GP1266 and GP1260(AR168G) IP phones were designed specially by Palmmicro, they used same phone case. Because of different hardware design, the 2 IP phones can not use the same firmware to upgrade. GP1266 use upgrade files like gp1266_sip_us_018000.bin and AR168G(GP1260) use files like ar168g_iax2_us_018000.bin. It does not matter when you try to upgrade one phone with another's firmware, web interface will tell you upgrade failed, and TFTP command line update will go very fast, about 9 seconds, while a normal update takes about 24 seconds.
There are a lot of methods to decide which hardware type it is. The most easy way is to check the back light when you pick up: GP1266 uses green back light, and AR168G(GP1260) uses blue back light.
Compared with GP1260(AR168G), GP1266 has much lower hardware board noise and more steady function keys.
Unlike AR168F, GP1260(AR168G) and GP1266 both used 132x64 dot matrix display, the software can be more fancy than 2x16 character LCD display, and can support Chinese and other special chars not included in standard ASCII table. They also support more keys, for example, max 10 speed dial.
