0.63 changes:
  1. Added OEM_10BASET compile option to use only port 1 of KSZ8842 and force it in 10BaseT.
  2. Changed RoIP COR/PTT signals exchange method from RFC 2833 to SIP MESSAGE text.
  3. Added 30 seconds time out in both sides of RoIP to limit the max continuing talking time of one side.
  4. Fixed a DNS bug when using direct IP call.
  5. Tools in SDCC\bin compiler upgraded from VS2008/VC9 to VS2013/VC12, need to install the x86 version of Visual C++ Redistributable Packages for Visual Studio 2013.
  6. Added OEM_JOSN compile option to keep using RFC 2833 for RoIP COR/PTT signals exchange method.
  7. Added hardware type AR168RT for RoIP module with KSZ8842.
  8. By using PHY linked status, removed the 2 seconds wait time in the init process of KSZ8842.
  9. Added SYS_MINI_RUN in version.h, the RTP first Minu Run for RTL8019AS will only work when it is defined.

Known problems:

    1. French version display error, other language except English and Chinese might have errors too.
    2. SDCC\bin\Manager.exe can not display GB2312 Chinese debug messages when running on Windows 7.
